## How To Get Started

```
# Clone repository
$ git clone https://github.com/Jalapina/friendZone.git

# Go into directory
$ npm install

# Go into client folder
$ npm install
$ ng build -w

# Go back to main directory 
$ nodemon server.js

Make sure to run your mongod server
Server will be running on port 8000
```
